# MKP Optimization Benchmark Study

A benchmark-driven optimization study comparing:

- LP Relaxation
- Rounded LP Heuristics
- Integer Programming (IP)

across multiple Multidimensional Knapsack Problem (MKP) benchmark instances.

---

# Project Objective

The project investigates how relaxed optimization methods and heuristic rounding strategies compare against exact integer optimization in terms of:

- Approximation quality
- Operational feasibility
- Constraint violation behavior
- Statistical performance consistency

---

# Methods Used

## Optimization Models
- Linear Programming (LP Relaxation)
- Rounded LP Heuristic
- Integer Programming (IP)

## Statistical Analysis
- Friedman Test
- Nemenyi Post-Hoc Analysis

## Benchmark Metrics
- Approximation Error
- Constraint Violation Ratio
- Feasibility Analysis

---

# Key Insights

- LP Relaxation produced optimistic approximation behavior due to relaxed feasibility conditions.
- Rounded LP remained statistically similar to LP relaxation in several benchmark environments.
- Integer Programming consistently maintained exact feasible optimization performance.
- The study revealed an important tradeoff between optimization quality and operational feasibility.
